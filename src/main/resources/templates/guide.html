권장 프로젝트 구조

│   └── test/
├── pom.xml 또는 build.gradle
└── README.md

3. 배포 프로세스

배포 전 모든 테스트가 통과되었는지 확인하세요.
배포 단계
  1. 프로젝트 빌드
  2. 테스트 실행
  3. 배포 파일 생성
  4. 서버 업로드
  5. 서비스 시작

4. 문제 해결

  • 의존성 충돌 확인
  • Java 버전 확인
  • 빌드 로그 확인

다음 사항들을 확인해보세요:

  • 포트 충돌 여부
  • 환경 변수 설정
  • 로그 파일 확인
